# Agenda Escolar – Calculadora de Calificaciones

Este proyecto es una aplicación web sencilla desarrollada con **HTML, CSS y JavaScript**, que permite capturar calificaciones de un estudiante y generar diferentes resultados de forma dinámica.

## Funcionalidades

- Ingresar nombre y calificaciones.
- Mostrar las calificaciones en una tabla.
- Calcular el **promedio general**.
- Encontrar la **calificación más alta** junto con la materia correspondiente.
- Mostrar las **materias reprobadas** (si existen).
- Validación de datos (solo valores entre 0 y 10).

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript (DOM, validaciones y cálculos)
